# Схема подключения сервоприводов

## Обозначения
- **FL** - Front Left (Передняя левая нога)
- **FR** - Front Right (Передняя правая нога)  
- **BL** - Back Left (Задняя левая нога)
- **BR** - Back Right (Задняя правая нога)

## Части ноги (5-звенный механизм)
Каждая нога имеет 2 сервопривода:
1. **SHOULDER (Плечо)** - верхний сустав, поднимает/опускает ногу
2. **HIP (Бедро)** - нижний сустав, сгибает/разгибает ногу

```
        SHOULDER
           |
           |
          HIP
           |
           |
         FOOT
```

## Нумерация каналов PCA9685 (0-15)

### Текущая конфигурация (4 серво - передние ноги):
| Канал | Нога | Часть     | Описание                    |
|-------|------|-----------|----------------------------|
| 0     | FL   | SHOULDER  | Передняя левая - плечо     |
| 1     | FL   | HIP       | Передняя левая - бедро     |
| 2     | FR   | SHOULDER  | Передняя правая - плечо    |
| 3     | FR   | HIP       | Передняя правая - бедро    |

### Полная конфигурация (8 серво - все 4 ноги):
| Канал | Нога | Часть     | Описание                    |
|-------|------|-----------|----------------------------|
| 0     | FL   | SHOULDER  | Передняя левая - плечо     |
| 1     | FL   | HIP       | Передняя левая - бедро     |
| 2     | FR   | SHOULDER  | Передняя правая - плечо    |
| 3     | FR   | HIP       | Передняя правая - бедро    |
| 4     | BL   | SHOULDER  | Задняя левая - плечо       |
| 5     | BL   | HIP       | Задняя левая - бедро       |
| 6     | BR   | SHOULDER  | Задняя правая - плечо      |
| 7     | BR   | HIP       | Задняя правая - бедро      |

## Визуальная схема робота (вид сверху)

```
      ПЕРЕДНЯЯ СТОРОНА
    FL (0,1)    FR (2,3)
        O          O
        |          |
        |          |
    ================
    |   РОБОТ      |
    ================
        |          |
        |          |
        O          O
    BL (4,5)    BR (6,7)
      ЗАДНЯЯ СТОРОНА
```

## Калибровка
Для каждого серво нужно определить:
1. **Минимальный угол** - ограничение движения
2. **Максимальный угол** - ограничение движения  
3. **Offset (смещение)** - корректировка нулевого положения
4. **Направление** - нормальное или инверсное

## Тестирование
Используйте программу `servo_calibration` для определения:
- Какой канал управляет какой частью ноги
- Диапазон движения (min/max углы)
- Необходимое смещение (offset)
